# Data-Analysis-using-Pandas

## Overview

This project analyzes student usage of Generative AI tools and their impact on academic performance using Python and Pandas.

## Objectives

* Load and analyze dataset
* Clean and process data
* Generate summary statistics
* Visualize key trends
* Extract meaningful insights

## Technologies Used

* Python
* Pandas
* Matplotlib
* VS Code

## Features

* Data Cleaning
* Missing Value Analysis
* Student Major Analysis
* GPA Analysis
* Burnout Risk Analysis
* Data Visualization

## Visualizations

* Students by Major Category
* Year of Study Distribution
* Primary Use Case of AI
* Burnout Risk Levels
* GPA Comparison

## How to Run

Install dependencies:

pip install pandas matplotlib

Run:

python main.py

## Key Insights

* Different majors use AI tools differently.
* AI usage varies by year of study.
* GPA changes can be analyzed before and after AI adoption.
* Burnout risk levels vary among students.

## Future Scope

* Machine Learning Prediction Models

## Conclusion

This project demonstrates how data analysis techniques can be used to understand student AI usage patterns and their impact on academic performance.
